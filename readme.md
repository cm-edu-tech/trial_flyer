### Parameters

```
name=<student name>
course=<course name>
instructor=<instructor name>
datetime=<date and time>
mode=<ZoomURL or Organization Name> (e.g., name should be: Irvine, Seattle, Arcadia, Diamond Bar, WL, San Diego)
```

Examples：

1. Zoom

```
https://cm-edu-tech.github.io/trial_flyer/index.html?name=John%20Smith&course=Roblox%201&instructor=Kevin%20Gee&datetime=2024%E5%B9%B412%E6%9C%8812%E6%97%A5&mode=https://zoom.ui/j/123456789
```

2. 线下

```
https://cm-edu-tech.github.io/trial_flyer/index.html?name=John%20Smith&course=Roblox%201&instructor=Kevin%20Gee&datetime=2024%E5%B9%B412%E6%9C%8812%E6%97%A5&mode=Irvine
```
